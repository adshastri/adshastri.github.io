Portfolio

Robota

A robot to assist special needs students in communicating with teachers

Robota uses Clarifai's video analysis tools to recognize when a student is frustrated and then moves over to them to prompt them for an explanation. It then uses IBM Watson's tools to transcribe and analyze the message and uses Nexmo to send a text to the teacher.

Built and demoed at TechCrunch Disrupt 2017

Alexa Waiter

A cross-platform app to make restaurants more efficient

This project helps restaurants work more efficiently by automating the ordering process. An Alexa skill is used to take orders, and customers can see live updating receipts and orders. Restaurants, on the other side, can see their revenue total or register for this service.

Built and demoed at HackPrinceton Spring 2017 - Top 10 and Best Financial Hack

Finish

A cross-platform app to help people track their habits

This React Native app tracks people’s habits and sends push notifications daily. It is built with a MeteorJs backend that integrates Facebook + Google OAuth and saves habits to a MongoDB database.

Built Fall 2016-present, on iOS app store

Here to There

A project to help navigate the Rutgers Bus system

Here to There builds on top of existing Rutgers applications and APIs by integrating geocoding and an Alexa skill. Given two locations on Rutgers, it finds the closest bus stop to each location and the appropriate bus to take between the two.

Built at HackRU Fall 2016

VR Chess

A multiplayer, voice controlled virtual reality chess game

This game is an Android app that uses Unity3D and the Google Cardboard to show the user a 3D chessboard and interact with a MeteorJs server to connect with an opponent. It accepts voice commands to control pieces on the chessboard (currently incomplete).
